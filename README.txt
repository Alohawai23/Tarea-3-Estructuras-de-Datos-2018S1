Instrucciones de compilado
Para compilar el problema  ingresar la siguiente linea en la terminal: gcc problema1.c -o problema1
Luego para lograr correr el programa, se debe ingresar los datos mediante un archivo de texto de manera standard, lo cual significa que se debe ingresar la siguiente linea en la terminal:
./problema1 < archivo.txt 
Donde archivo.txt sera el archivo de entrada de datos.
